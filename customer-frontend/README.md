# Restaurant Customer App\n\nA customer-facing React application where users can browse the restaurant menu and place orders.\n\n## Features\n\n- Browse available menu items\n- Search menu items by name or ingredients\n- Filter by category (Appetizer, Main Course, Dessert, Beverage)\n- Add items to cart with quantity management\n- Place orders with customer name and table number\n- Real-time cart updates\n- Responsive design\n\n## Setup\n\n1. Install dependencies:\n```bash\nnpm install\n```\n\n2. Create `.env` file:\n```env\nREACT_APP_API_URL=http://localhost:5000/api\n```\n\n3. Start the application:\n```bash\nnpm start\n```\n\nThe app will run on http://localhost:3001 (or next available port)\n\n## Usage\n\n1. Browse the menu items displayed in cards\n2. Use search to find specific items\n3. Filter by category using the dropdown\n4. Click \"Add to Cart\" to add items\n5. Click the cart icon to view/modify your order\n6. Enter your name and optional table number\n7. Click \"Place Order\" to submit\n\n## API Integration\n\nThe app connects to the same backend API as the admin dashboard:\n- `GET /api/menu?availability=true` - Get available menu items\n- `GET /api/menu/search?q=query` - Search menu items\n- `POST /api/orders` - Place new order\n\n## Technologies\n\n- React 18+ with TypeScript\n- Tailwind CSS for styling\n- Axios for API calls\n- Context API for cart state management\n- Lucide React for icons
